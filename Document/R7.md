算法有问题。
单考虑绿色，96个颜色绝对够用了，而且还有微调值做后处理，它怎么说也能覆盖整个色域（饱和度、明度全覆盖的色库）
所以，算法重点是不同色的区分和相似色的合并和微调值的计算

需要根据人眼剔除掉不用的颜色（可调整覆盖的值）
先作色相/分辨度函数。以密度转换做成1d的函数，表示在某一色相进行变化，对人眼造成的影响多么显著
再作色相/饱和度函数。做成1d函数，表明在某一色相下，饱和度变化对人眼造成的影响多么显著
最后以这二者作2d函数达成对颜色的计算机数值变化率到人眼生理变化率的转换。以更好地优化色库的算法

现在我们有了这个2d函数。

我们来谈谈如何利用这个函数
比如一个树，那么我们先采样出来它的所有颜色，例如棕色、墨绿、草绿、杂色绿1、杂色绿2
然后把它们加入到色库。我们不根据数量进行剔除，而是颜色相近性，以及在这之前，我们还需要计算微调值。
为了方便管理，我们把所有值都归255，即让rgb的最大值为255，其他也受此影响调整。比如颜色（100, 200, 150）
Max = 200
缩放因子（微调值的倒数） = 255 / 200 = 1.275
就变成
(128, 255, 191)
这样比较方便微调值的使用。微调值的意义区间可以被限制在0-2之间（比1大的时候会提升白度，而归255化正好会让微调值为2的时候纯白）。
虽说是有损压缩，但或许这甚至有利于我们，这样可以二次优化相近颜色

微调值（我们命名为Ko）的定义：缩放因子的倒数
Ko = 缩放因子的倒数 = max / 255
其中 max 是原始 RGB 的最大值。
Ko 的值域范围 0-1，预留到 2 的空间用于后处理

如果杂色绿1和草绿很接近，那么我们就把它们合并为一个颜色（取中间值），然后以此类推。我们最后会剩下仅仅几种颜色和合理的微调值。
然后有了杂色绿1归255后的颜色，那么如果另一个杂色绿2归255后和它值几乎一样（这里用近似的话要极小容差值，不然丢精度太多了），就不再存新色库色，只是两个杂色绿有不同的微调值。

注意：要先作近似色剔除再算微调值

所以色库里应该都是255化的颜色

对于其他模型。比如我们加入了藤蔓模型。它在处理的时候应该会先剔除近似色。算微调值归255，看看自己有的颜色色库里有没有相近的，如果有，就把它身上的相近颜色用色库的。如果没有，就可以将自己的颜色作为新颜色加入色库。

总结全部流程：

新模型：作近似色剔除：算微调值归255：查色库找相近值：使用色库的相近值（不作平均，直接用，不然会造成数据污染，不用担心颜色单一，因为有近似色兜底存新颜色）：存色库和自己的微调值


关键附录：

注意：低饱和度时，人眼对饱和度变化最敏感；高饱和度时，最不敏感。
计算变化值：
比如两个颜色变化值（色相和饱和度的二合一变化值），要用方程

ΔP = √[ (ΔH × K_h(h))² + (ΔS × K_s(h) × (1 - s)^0.6)² ]
ΔP	人眼感知变化量	≥ 0	输出值
ΔH	色相变化	0-180°	颜色差值
ΔS	饱和度变化	0-1	颜色差值
h	起始色相	0-360°	颜色值
s	起始饱和度	0-1	颜色值
K_h(h)	色相敏感度	0-1	GMM拟合
K_s(h)	饱和度敏感度	0-1	GMM拟合

然后需要一个变化值临界值Ji，这是个经验数值，性质是，如果颜色变化值比Ji大，就认为是显著变化，存两个颜色。如果小就取均值合并

我们现在先取 Ji 为0.05（经验实验数值0.05）

计算色相用
设：
Max = max(R, G, B)
Min = min(R, G, B)
Delta = Max - Min
色相角 Hue 的计算：
如果 Delta == 0:  Hue = 0 (无彩色，如灰色，色相无定义)
否则：
    如果 Max == R: Hue = 60 * (((G - B) / Delta) mod 6)
    如果 Max == G: Hue = 60 * (((B - R) / Delta) + 2)
    如果 Max == B: Hue = 60 * (((R - G) / Delta) + 4)

Keyness2D 的作用
这个类计算的是：在某个颜色位置，人眼对颜色变化的敏感度。
hpp：

// keyness_2d.hpp
// 计算颜色感知敏感度权重
// 值越高 = 人眼越敏感 = 需要越多颜色锚点

#pragma once
#include <cmath>
#include <array>

class Keyness2D {
public:
    Keyness2D() = default;
    
    // 输入: 色相(0-360°), 饱和度(0-1)
    // 输出: 敏感度权重(0-1)
    double keyness(double hue_deg, double saturation) const {
        const double PI = 3.141592653589793;
        double hue_rad = hue_deg * PI / 180.0;
        
        // 包装到 [-PI, PI]
        if (hue_rad > PI) hue_rad -= 2 * PI;
        if (hue_rad < -PI) hue_rad += 2 * PI;
        
        double hue_key = gmm1d(hue_rad, hue_weights, hue_means, hue_covs);
        double sat_sens = gmm1d(hue_rad, sat_weights, sat_means, sat_covs);
        double sat_factor = std::pow(saturation, 0.6);
        
        double result = hue_key * sat_sens * sat_factor / 0.00556946;
        
        return std::min(1.0, result);
    }
    
private:
    static constexpr int N = 8;
    using Array = std::array<double, N>;
    
    // 色相敏感度 GMM (从锚点密度拟合)
    static constexpr Array hue_weights = {
        0.11132848, 0.13530582, 0.10849456, 0.11214354,
        0.22830474, 0.09697763, 0.12251557, 0.08492966
    };
    
    static constexpr Array hue_means = {
        6.96195504, -2.89108923, 11.31023104, 0.57595815,
        4.04752825, -5.55301231, 9.69434493, -1.34614132
    };
    
    static constexpr Array hue_covs = {
        0.38831633, 0.43726528, 0.33392143, 0.32444862,
        1.07510566, 0.19457601, 0.35457794, 0.15187700
    };
    
    // 饱和度敏感度 GMM (从实验数据拟合)
    static constexpr Array sat_weights = {
        0.09713084, 0.13469279, 0.11840812, 0.15963278,
        0.10077283, 0.11777569, 0.13832214, 0.13326481
    };
    
    static constexpr Array sat_means = {
        -5.28501930, 4.79250700, 9.46615962, -0.27033550,
        2.54706985, 6.65366799, -2.55780595, 11.54815262
    };
    
    static constexpr Array sat_covs = {
        0.51787753, 0.71454204, 0.81614231, 0.80506982,
        0.91408268, 0.72683746, 0.93444933, 0.41576665
    };
    
    double gmm1d(double x, const Array& w, const Array& m, const Array& v) const {
        const double PI = 3.141592653589793;
        double density = 0.0;
        
        for (int i = 0; i < N; ++i) {
            double diff = x - m[i];
            double exp_term = -(diff * diff) / (2.0 * v[i]);
            density += w[i] * std::exp(exp_term) / std::sqrt(2.0 * PI * v[i]);
        }
        
        return density;
    }
};
