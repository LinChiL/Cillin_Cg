算法有问题。
我说实话，单考虑个绿色，96个颜色绝对够用了，而且还有个叫微调值的东西呢，它怎么说也能覆盖整个绿色域（饱和度、明度全覆盖的绿色库）
所以，算法重点是不同色的区分和相似色的合并和微调值的计算

还是拿最经典的绿色域来说。


计算色相用
设：
Max = max(R, G, B)
Min = min(R, G, B)
Delta = Max - Min
色相角 Hue 的计算：
如果 Delta == 0:  Hue = 0 (无彩色，如灰色，色相无定义)
否则：
    如果 Max == R: Hue = 60 * (((G - B) / Delta) mod 6)
    如果 Max == G: Hue = 60 * (((B - R) / Delta) + 2)
    如果 Max == B: Hue = 60 * (((R - G) / Delta) + 4)
